# CLAUDE.md

## Standing instruction: keep docs current

After any change that alters the architecture, adds a feature, or fixes a bug:

- **Always** add a bullet to `CHANGELOG.md` describing what changed.
- Update `ARCHITECTURE.md` if the stack, auth model, data flow, or file structure changed.
- Update `README.md` if the setup steps, env vars, project structure, or security notes changed.
- Update `TODO.md` if the change completes an item or reveals a new one.
